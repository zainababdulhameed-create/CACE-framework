# CACE-Framework

Confidence-Adaptive Chaos Encryption with Deep Feature Classification for Secure Medical Image Transmission over IoT Networks

## Overview
A unified pipeline integrating ResNet-50 deep feature classification with 4D chaotic Chen encryption. The SVM classifier confidence score governs encryption intensity (1-5 rounds), yielding 20.2% computational savings.

## Dataset
Brain Tumor MRI Dataset: https://www.kaggle.com/datasets/masoudnickparvar/brain-tumor-mri-dataset
